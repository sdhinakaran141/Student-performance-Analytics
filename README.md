 Student-Performance-Analytics

## Project Descriptions
This Project analyze student performance data using python, pandas, NumPy, and Matplotlib .It performs data cleaning, grade calculation, 
statistical analysis, visualization, and exports the results to CSV and Excel formats.


## Technology

-Python
-Pandas
-NumPy
-Matplotlib


## Dataset
Students.csv

## Features

-Data cleaning
-Grade Calculation
-Total calculation
-Average Calculation
-Department Analysis
-Bar Chart
-Line Chart
-Pie Chart
-CSV Export
-Excel Export
-Json Export
